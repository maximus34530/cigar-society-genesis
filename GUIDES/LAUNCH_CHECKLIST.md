# Launch checklist (Issue 25 — owner review)

Complete with the business owner before pointing the custom domain at production.

- [ ] **Home** — Copy, hero, CTAs, and video/poster look correct on phone and desktop
- [ ] **About** — Story, certifications, and imagery are accurate
- [ ] **Cigars** — Selection descriptions reflect what you want to promote
- [ ] **Events** — Dates/times and disclaimers match how you run events
- [ ] **Gallery** — Photos are current and approved for public use
- [ ] **Contact** — Phone, address, hours, and map pin are correct
- [ ] **Footer / nav** — Links and social URLs work
- [ ] **Legal / policy** — Age requirement (21+) and any disclaimers are acceptable
- [ ] **Go / no-go** — Owner approves launch

Tracking: file follow-up items as separate GitHub issues when changes are needed.
